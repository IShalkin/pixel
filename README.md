# pixel

A skill for turning one complex or abstract concept into a single
self-contained, interactive HTML page that teaches it through a concrete
SITUATION the audience steps through. Method-first: the situation and the visual
style are invented fresh for each concept, never cloned from an example.

See [`SKILL.md`](SKILL.md) for the full method, the borrowable techniques
(canvas text wrapping and clipping, the one-rig walk/dance figure, emotes,
parallax, NPC steering with goal-driven activities), and the verification
checklist.

## Layout

```
SKILL.md      the skill itself
reference/    five worked example pages (study the METHOD, do not clone)
assets/       optional sprite art (LPC); procedural drawing is the default
```
